### 2.字体
>**加粗**
>>`语法：**加粗**`

>*斜体*
>>`语法：*斜体*`

>***加粗斜体***
>>`语法：***加粗斜体***`

>~~删除线~~
>>`语法：~~删除线~~`

### 3. 代码块
>单行代码( `` 包裹, 两个反引号 )
>>`console.log(1)`

>多行代码 （三个 ` 包裹）
>>```js
>>    function fn () {
>>        console.log('fn')
>>    }
>>```

### 7. 有序/无序列表
>```
>1. 效果如下 - + *（序号内容必须有空格，上下级要有缩进）
>   1. 下级 
> 
>无序列表：（嵌套加tab即可）
>+ 无序1 （数字即有序列表）
>   + 无序2 （无序列表用 - + * 任何一种都可以）
>```
>>1. 效果如下 - + *（序号内容必须有空格，上下级要有缩进）
>>   1. 下级 
>>
>>无序列表：（嵌套加tab即可）
>>+ 无序1 （数字即有序列表）
>>  + 无序2 （无序列表用 - + * 任何一种都可以）
